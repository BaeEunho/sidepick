# SidePick Backend API

사이드픽 백엔드 API 서버

## 기술 스택
- Node.js + Express
- Firebase Firestore
- JWT Authentication

## 환경 변수
`.env` 파일에 다음 변수들을 설정하세요:
```
JWT_SECRET=your-secret-key
EMAIL_USER=your-gmail@gmail.com
EMAIL_PASS=your-app-password
```

## 실행 방법
```bash
npm install
npm start
```
